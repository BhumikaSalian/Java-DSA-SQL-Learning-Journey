9. Database Connection Program
File:
src/main/java/com/bhumika/DatabaseConnection.java
Code:
package com.bhumika;

import java.sql.Connection;
import java.sql.DriverManager;

public class DatabaseConnection {

    public static void main(String[] args) {

        String url = "jdbc:mysql://localhost:3306/JavaDemoDB";

        String username = "root";
        String password = "YOUR_PASSWORD";

        try {

            Connection con =
                    DriverManager.getConnection(
                            url,
                            username,
                            password);

            System.out.println(
                    "Database Connected Successfully!");

            con.close();

        } catch(Exception e) {

            e.printStackTrace();
        }
    }
}

10. Run Java Program
Compile:

Run:

Viva Questions:
1. What is Maven?
Build automation and dependency management tool.

2. What is Group ID?
Unique identifier of organization/project.

3. What is Artifact ID?
Project name.

4. What is pom.xml?
Project Object Model file containing dependencies and build configuration.

5. What is JDBC?
Java Database Connectivity API.

6. What is MySQL Connector/J?
JDBC driver used to connect Java with MySQL.

7. What is DriverManager?
Class that establishes database connections.

8. What is Connection?
Represents a session with the database.
